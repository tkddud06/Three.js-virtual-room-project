# Three.js를 이용한 방 만들기 프로젝트
>**2025-02 컴퓨터그래픽스 과목 개인 프로젝트**<br/>

## 프로젝트 소개

<img width="1423" height="801" alt="image" src="https://github.com/user-attachments/assets/22d5136b-1ff6-4c00-8f4a-03c205187807" />

자바스크립트 라이브러리인 Three.js를 활용하여 가상의 컴퓨터 방 씬을 제작해보는 프로젝트입니다.
모델링 툴, Three.js의 각종 기능들을 활용하여 현실감을 높였습니다.


## 개발 환경

- **Three.js**: r132 (0.132.0) (CDN)
- **IDE**: Visual Studio Code (with Live Server Extension)
- **모델링 툴**: Blender



## 핵심 내용

### 1. 사실적인 재질 구현 및 텍스처 매핑

* **커스텀 텍스처 제작**: 실제 사물을 촬영한 이미지를 바탕으로 텍스처를 제작하고, `TextureLoader`를 통해 매핑하여 현실감을 높였습니다.
* **PBR(Physically Based Rendering) 활용**: 금속 재질의 반사광(Metalness)과 거칠기(Roughness)를 조절하여 실제 가구와 전자제품의 질감을 재현했습니다.

### 2. 정교한 조명 설계 및 그림자 구현

* **다양한 광원 활용**: 전체적인 밝기를 위한 `AmbientLight`, 벽 장식장 조명과 자연스러운 조명효과을 위한 `SpotLight`, `RectAreaLight` 등을 적재적소에 배치했습니다.
* **그림자 최적화**: `ShadowMap` 설정을 통해 그림자의 해상도와 부드러움(Soft Shadows)을 조절하여 자연스러운 공간감을 연출했습니다.

### 3. Blender 모델링과 Three.js의 모델링 혼합 사용

* **모델링 혼합 사용**: 기본 도형으로 구현 가능한 벽면이나 바닥은 Three.js 내부 지오메트리를 사용하고, 복잡한 소품은 직접 Blender를 통해 모델링한 외부 모델 혹은 무료 모델링을 사용하여 렌더링 효율을 극대화했습니다.

### 4. 사용자 상호작용 및 카메라 제어

* **TrackballControls 적용**: 사용자가 마우스 드래그와 휠을 통해 방 전체를 자유롭게 회전하고 확대/축소하며 관람할 수 있는 기능을 제공합니다.
* **이벤트 핸들링**: `Raycaster`를 활용하여 방 안의 특정 사물(예: 모니터)을 클릭했을 때 반응하는 인터랙티브 요소를 포함했습니다. `AnimationClip`을 활용하여 조명 및 물체 회전 컨트롤이 가능하도록 하였으며 낮과 밤 변화, 카메라 자동 회전 핸들링 기능을 제공합니다.
